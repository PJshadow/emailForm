Description:
This is a very simple form, which has the option of attaching a .pdf file to it.

Installation:
Clone the repository. Then run "npm i" to install all the dependencies.

Environment:
In order to use it, a .env file is needed. Create it with the following data:
SMTP_USER=
SMTP_PASS=
SMTP_HOST=
If it doesn't work, you'll probably need to change some data within the index.js file, according to your email server.
